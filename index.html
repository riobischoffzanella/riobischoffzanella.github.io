    Course Project with Unreal 5

    As the final project of the course, they have asked for a complete level structure in Unreal, requiring the steps and methods you have learned to create it from start to finish. It was a great way to put in practice the tools I've learned throughout the course, like vistas, Level Diagrams, Level Maps, Level Design Documents, and more.

    Unfortunately, I did not apply enemies as it would require more that it was asked from the assignment. With a set game design with programmed enemies, though I could set them just fine, as intended and expressed in the video

    Tools

    • Engine: Unreal Engine 5
    • Engine Tools: Greyboxing
    • Engine Tools: Landscape mode
    • Pen and paper
    • Course length: 5 to 6 minutes


    Design Process and Goals

    • Make a engaging FPS level in Unreal to test out my skills in both using the engine, and what I've learned from theory classes.
    • Verticality
    • Clear path
    • THEME: Getting your wallet back and facing enemies throughout the way
